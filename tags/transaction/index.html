详解 IB 的 MOC 和 LOC 委托类型：收盘价交易的两种策略

2025年10月15日

在美股等成熟市场中，收盘价（Closing Price）具有重要的参考意义，它不仅是当天交易情绪的总结，也是许多指数计算、基金净值核算和投资组合估值的基准。因此，针对收盘价的交易需求应运而生。在盈透证券（Interactive Brokers, IB）的交易平台中，MOC（Market-on-Close，市价收盘） 和 LOC（Limit-on-Close，限价收盘） 是两种允许投资者在收盘时段执行交易的重要委托类型。
