不常用，但是很有用的git命令

重装系统导致windows上文件夹所有者信息变更

新版本git增加了安全性校验，提示不安全，导致无法继续操作

git config --global --add safe.directory
git config --global --add safe.directory "*"

新电脑保存账号密码信息

git config --global credential.helper store

如果保存的信息需要更新，先清理老的凭证

git config --system --unset credential.helper
