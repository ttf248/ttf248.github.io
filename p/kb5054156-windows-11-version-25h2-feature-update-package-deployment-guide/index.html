KB5054156：Windows 11 版本 25H2 功能更新（启用包使用指南）

2025年10月28日
阅读约 4 分钟
tianlong.xiang
English 日本語

更新本地的 windows 系统安装包，发现已经发布了 25h2 版本，本地的系统还停留在 24h2 版本，微软的更新补丁都安装了，但是本地的系统还没有升级到 25h2 版本。好奇中间少了什么步骤。

本文基于微软官方支持文档，梳理 KB5054156 更新的核心信息，帮助用户理解从 Windows 11 24H2 升级至 25H2 的关键要点。

更新概述

KB5054156是针对Windows 11 版本25H2的功能更新，核心通过“启用包”激活25H2版本新功能。其本质是利用Windows 11 24H2与25H2共享“通用核心系统”的特性——25H2的新功能已包含在24H2的最新月度质量更新中，但处于休眠状态，启用包则作为“主开关”激活这些功能。

应用对象

该更新仅支持运行Windows 11 版本24H2的设备，具体版本包括：

  • Windows 11 Enterprise and Education, version 24H2
  • Windows 11 Enterprise Multi-Session, version 24H2
  • Windows 11 Home and Pro, version 24H2
  • Windows 11 IoT Enterprise, version 24H2

启用包的核心优势

相比传统功能更新，启用包的核心价值在于减少更新停机时间：

  • 无需复杂下载与安装流程，仅需一次设备重启即可完成从24H2到25H2的升级；
  • 升级后设备可立即使用25H2版本的新功能，无需等待完整系统更新。

更新获取方式

KB5054156通过三种渠道发布，不同渠道的可用性与操作步骤如下：

发布渠道 可用性 下一步操作
Windows Update 可用 自动下载并安装，功能更新显示为“Windows 11，版本 25H2”，无需手动触发
Windows 更新目录 不可用 无，此更新仅通过Windows Update和WSUS渠道获取
Windows Server Update Services (WSUS) 可用 需配置以下参数实现自动同步：
1. 产品：Windows 11
2. 分类：升级
更新名称为“Windows 11，版本 25H2”

升级先决条件

在应用KB5054156更新前，必须满足两个条件：

  1. 设备当前运行的系统版本为Windows 11 版本24H2（不支持从更低版本直接升级）；
  2. 已安装2025年8月29日发布的KB5064081累积更新（OS 内部版本 26100.5074，含预览版及更高版本）。

重启与更新替代说明

  1. 重启要求：应用KB5054156更新后，必须重启设备才能完成25H2功能的激活；
  2. 更新替代：此更新不会替换任何此前发布的Windows更新，无需担心覆盖历史更新文件。
