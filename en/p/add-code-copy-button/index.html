Main body

I spent over half an hour looking at ouuan’s code, but I still don’t really understand how to add the copy button

Time passes, and after a month, I thought of this again

Since I didn’t understand this assignment, let’s just copy another one; I should be able to figure it out eventually. The search results were surprisingly good – there was even a post in the hugo official forum mentioning how to add a copy button. After checking it out, the logic is clear. Then something unexpected happened: when I looked at the site, the code block style rendered by even differed from the description in the documentation. This part is a bit complicated, so here’s a brief record.

Because I barely understood front-end development, whenever I didn’t understand something, I would open the browser’s “Inspect Element,” analyze the code, and rely on the style information on the right. Slowly, I began to understand the logic. I didn’t understand JavaScript, so I added some logs in the console. There were many things I didn’t understand, but I calmed down, slowly sorted out and broke down the logic, and eventually found a way forward.

  1. There are multiple pre nodes, referring here to a single code block. The theme renders line numbers, resulting in two copy buttons appearing.
  2. I want code highlighting in the theme, but I’m not familiar with how to set it up in this theme
  3. I looked at the documentation on the Hugo website, and I understand that there’s a markup setting to control code highlighting
  4. Adjusting the configuration file, it’s always been wrong, and the rendering doesn’t match expectations
  5. I found these settings for pygmentsOptions, continued researching, adjusted the settings, and first removed the line numbers
  6. Configure custom CSS stylesheets and configure custom JS scripts
  7. Since we’ve gotten things to this point, it suddenly occurred to me that I saw a color scheme recently that was pretty good. Let’s change the button style: going with a Chinese-style celadon blue.

I spent four hours on it, and when I saw that sentence, I thought it was funny – how could it take so long? Then I looked at the time: three hours
