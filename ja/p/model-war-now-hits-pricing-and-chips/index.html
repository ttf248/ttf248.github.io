このモデル競争は、すでに価格やチップのレベルまでエスカレートしている。

2026年04月24日
読了時間: 15分
Codex (GPT-5)
中文 English

今夜、モデルのメッセージを追っていると、本当に目が点になった。

公式サイトのタイムラインを見ると、今回の集中更新は連続していることが分かります：2026-04-20 にMoonshotがKimi K2.6をトップページに掲げました。2026-04-22にはXiaomiが正式にMiMo-V2.5とMiMo-V2.5-Proをリリースし、2026-04-23にはOpenAIがGPT-5.5を公開し、API料金も同時に引き上げられました。そして2026-04-24にはDeepSeekがまたV4 Previewを推しました。ついでに言うと、グループチャットでよく話題になる「Xiaomi 2.5」というのは、厳密に言えばぼんやりとしたコードネームではなく、Xiaomi MiMo-V2.5 / V2.5-Proのことを指します。

私の現在の判断は非常に明確です。今回の波は単なるモデルのリリースブームではなく、3つの要素が同時にぶつかり合っています――すなわち、**モデル能力、API価格設定、チップスタックの支配権（または所有権）**です。これらの一つだけを語る人は、基本的に視点が偏りがちです。この三つの要素が絡み合うからこそ、大規模言語モデル（LLM）という分野はこれほどまでに過熱しているのです。

OpenAＩがハイエンドな価格帯を引き上げた

今回の最も目立つのは、GPT-5.5 のリリースそのものではなく、その価格シグナルだ。

OpenAIが『2026年4月23日』に公開した公式価格ページには、GPT-5.5の標準プランでの入力が$5 / 1M tokens、出力が$30 / 1M tokensであると明記されています。一方、GPT-5.4はそれぞれ$2.5 / 1Mおよび$15 / 1Mです。これは「ほぼ倍増」ではなく、直接倍増です。

正直、この手順はかなりキツイですね。

なぜなら、主要ベンダーが自らハイエンドな価格帯を引き上げると、市場の焦点はもはや単なる「誰がより強いか」という単純な比較ではなく、「どの強さが、この価格に見合うのか（価値があるのか）」という問題へと急速に変わるからです。 これは他のプレイヤーにとって非常に現実的な攻め口を与えます。OpenAIを完全に凌駕する必要はなく、コーディング、エージェント、長時間のタスク安定性といった高頻度なシナリオにおいてコストパフォーマンスの高さを示せば、十分に（大きな

国産はこの波、もはや単なる追い上げではない

この数日間の動向を「皆がOpenAIに追随して新しいモデルを出す」とだけ捉えると、この波の強さを過小評価しすぎかもしれません。

Kimi K2.6 の公式サイトと技術ブログによると、重点は一般的なチャットではなく、長尺なコーディング、Agent Swarm、継続的な実行に移っています。Moonshotが提供するケーススタディでは、モデルは4,000回以上のツール呼び出しや12時間以上にわたる連続実行をこなすことが可能です。APIプラットフォームのホームページに記載されている公開価格は、キャッシュが「$0.16 / MTok」、入力が「$0.95 / MTok」、出力が「$4.00 / MTok」です。ベンダーによる自社テストの結果を完全に信用することはできませんが、この価格設定のスタンスは非常に明確です。彼らは最高のプレミアムを争うのではなく、agentic codingというレイヤーを獲得しようとしているのです。

Xiaomi側も同様です。2026-04-22 にリリースされた MiMo-V2.5 と MiMo-V2.5-Pro は、主な焦点が「チャット体験」ではなく、agentic capability、multimodality、そして long horizon coherence に置かれています。MiMo-V2.5 は、1M context、マルチモーダル、Agentをトップページファーストビューに直接配置しました。一方、V2.5-Pro は、ベータ版の提供開始、価格据え置きであることを強調していますが、複雑なソフトウェア工学と長期間のタスク一貫性は引き続き引き上げられています。このシグナルは非常に重要です。現在、新しいモデルが獲得を目指しているポジションとは、「ツールチェーンに組み込めるか」「長期的なタスクを実行できるか」「実際に動作するか（役に立つか）」であるということです。

DeepSeekだとさらに明らかです。2026年4月24日のV4 Previewでは、1M context、V4-Pro、V4-Flash、およびAPIがその日に利用可能であることが直接公開されました。より重要なのは価格です。公式ドキュメントによると、deepseek-v4-proは入力$1.74 / 1M、出力$3.48 / 1Mであり、deepseek-v4-flashはさらに安く、入力$0.14 / 1M、出力$0.28 / 1Mです。この価格をGPT-5.5と比較すると、全く違う印象を受けます。

これはもはや「君が最先端モデルを持っているなら、僕にもある」というレベルではありません。

これは：

  • 高価格帯を引き上げようとするところがある；
  • オープン性、長尺なタスク処理、ツールコールといった機能でミドル〜ハイエンドの領域を狙っているところにいる；
  • 価格設定自体が無視できないレベルに直接落としている人もいます。

DeepSeekとHuaweiという連携に関する噂は、最も食い違いやすい（または誤解しやすい）線だ

あなたが挙げられたもう一つの点、そしてまさに今夜最も誤解が生じやすいところもそこです。それは、DeepSeek V4がすでにファーウェイのチップに対応しているのか、それとも下半期まで待つ必要があるのかという点についてです。

私が今回入手した公開情報に基づくと、より正確な言い方は以下の通りです：現在はすでにサポートされており、「下半期」というのは、より大規模に展開した後もコストが引き続き低下していくことを指します。

DeepSeekの公式な2026-04-24付のニュースページには、V4 Previewがすでにリリースされ、APIが当日利用可能である旨が記載されています。さらに、同日付けでHuaweiからの関連する公的発表においても、Ascendのsupernode製品ライン全体が既にDeepSeek V4をサポートしていることが明確に述べられています。したがって、「下半期にならないと完全に適応できない」という主張は、少なくとも本日確認できる公開情報とは矛盾しています（または「一致していません」）。

「下半期」に関連する真の意味は、別のレベルの話です。Huaweiによると、Ascend 950 supernodesが2026年後半に大規模に量産展開された後も、V4-Proの価格は大きく下がるということです。つまり、下半期というのは「ようやく動くようになる」という話ではなく、「動くことは既に確立された事実であり、次に注目すべきは、規模化した後も価格がさらに引き下げられるかどうか」ということです。

この違いは大きいです。

前者は技術的に成り立つか、後者は商業的にスケールできるか。

むしろ、これが今夜最も注目すべきシグナルだと感じています。なぜなら、DeepSeekのようなレベルのモデルが、本当に华为 Ascendというラインと安定的に結びつき始めた場合、大規模モデルの競争はもはや単にモデル企業同士の問題ではなくなり、トレーニング、推論、チップ、クラスター、価格設定が一連で連携するものとなるからです。このエンドツーエンドな全工程を確立できる者が、より強い発言権を持つことになるでしょう。

なぜ私が「分野はすでに過熱している」と言うのか

以前は皆、モデル戦争について語る際、ランキングやパラメータの数、ベンチマークスコア、そして「また進化した」という言葉に留まりがちでした。しかし、今は状況が変わりました。

現在の競争は、少なくとも以下の三つの層（レベル）で同時に展開されています：

第一の層は、モデル自体が「打てる」能力、特にコーディング、エージェント、ロングコンテキストといった、ワークフローに真に組み込める能力を持つことです。

二つ目ですが、料金設定が納得できるものである必要があります。なぜなら、モデルが実際のビジネスを受け付け始めると、トークンコストは抽象的な数字ではなく、月末の請求書となるからです。

第三層では、その基盤となるどのチップセット、どのクラスター、どのソフトウェアスタック上で動作しているかが、持続的な価格引き下げが可能か、安定した供給が維持できるか、地政学的およびサプライチェーンの制約を突破できるかという点に影響し始めます。

この3層が同時に巻き込まれると、真の白熱化となります。

だから、今となっては「どの会社が一番か」って聞くのがあまり気にならなくなってきた。あの質問はどんどん面白みがなくなってくるよ。

より役立つ質問は次のとおりです：

  • 抱えているタスクに対し、本当に GPT-5.5 のような高価格なモデルにお金を払う価値があるのか。
  • あなたの長尺なコーディングやAgentワークフローは、Kimi K2.6、MiMo-V2.5、DeepSeek V4 のような、より安価でありながら明らかな性能が高い（aggressive）モデルに任せられるか。
  • あなたのビジネスには、国産の算力、コンプライアンス、ローカルな展開（ローカライゼーション）、またはサプライチェーンの安定性に関する硬い制約はあるか。

ここが現実世界の選定です。

結局のところ、この白熱化は「発表会が増えた」というだけの単純な話ではありません。本当に市場の温度が上がっていると感じられるのは、トップメーカーが価格を上げることや、追随企業が本格的に競り合ってくる姿勢が見られること、さらには半導体メーカーまでが直接参入し、モデル技術

下半期に入り、私がより注目しているのは、誰がどれほど高いスコアを出せるかという点ではなく、**「能力」「コスト」「サプライチェーン」**のこの三つの要素を同時にいかに最適に両立させられるか、という点です。

これ（を）実現できるのだから、一時的なもの（はかないもの）ではない。
