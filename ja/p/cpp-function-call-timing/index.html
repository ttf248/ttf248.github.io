int add_one(int input) { return input + 1; }

bool validate_vector_add_one(std::vector<int> const& input_vector,
                             std::vector<int> const& output_vector)
{
    bool is_valid{true};
    for (size_t i{0}; i < input_vector.size(); ++i)
    {
        if (output_vector.at(i) != input_vector.at(i) + 1)
        {
            is_valid = false;
            break;
        }
    }
    return is_valid;
}

検証関数 validate_vector_add_one の説明:

この関数は、入力ベクトルと出力ベクトルの両方を引数として受け取り、入力ベクトルの各要素に1を加算したものが、出力ベクトルに対応する要素と等しいかどうかを検証します。 もし等しくない場合は false を返し、そうでなければ true を返します。

入力ベクトル: 整数値のベクタ 出力ベクトル: 整数値のベクタ
