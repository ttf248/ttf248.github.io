정적인 블로그 테마는 주로 해외 템플릿을 기반으로 수정 및 조정하며, 중국어 콘텐츠의 레이아웃에 대한 고려가 충분하지 않다

본문

반 달 전, 블로그 스타일 시트를 조금 조정했는데, 오랫동안 백엔드 서비스 개발만 해와서 프론트엔드는 완전 초보입니다. 앞뒤로 반나절을 끙끙거렸는데 디자인이 별로 마음에 들지 않았습니다. 문득 떠올라 자주 보는 기술 블로그인 infoq, 오픈소스 중국의 스타일이 정말 좋다고 생각했습니다. 이걸 조금 가져다 쓸 수 있을까요? 잠시 소스 파일을 살펴봤지만 관련된 요소를 찾으려니 머리가 엉망이 되었습니다.

프론트엔드 개발자 친구들은 여기서 웃을지도 모르겠네. 특정 요소를 찾는 것조차 모르는구만. 모르는 건 괜찮아, 주말에 시간도 많으니 잠시 멈춰서 생각해보니, 예전에 python 크롤러를 만들 때 비슷한 걸 사용했던 것 같아.

요소 검토

맞아요, 바로 브라우저 자체의 요소 검사 도구인데, 스타일 시트 복사나 특정 요소 찾기 같은 것들은 금방 할 수 있습니다. selector로 요소를 찾고, hugo에서 user define css를 새로 만들 수 있습니다.
