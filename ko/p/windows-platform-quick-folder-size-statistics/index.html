리눅스 플랫폼은 간단해서 du -sh * 한 줄의 코드로 끝내 버릴 수 있다. 윈도우는 어떨까? 디스크가 많고, 파일을 정리하고 싶을 때, 파일이 많은 상황에서 시스템 내장 리소스 관리자는 폴더 크기를 통계적으로 계산하는 속도가 너무 느려서 포기하고 싶게 만든다.

Everything

윈도우 플랫폼에서 개발하는 분이라면, 직접 사용해 보지 않더라도 친구에게 들어봤을 Everything이라는 프로그램이 있을 겁니다. 검색 속도는 시스템 자체의 파일 탐색기보다 훨씬 빠릅니다. 시스템 수준에서 빠른 파일 인덱싱을 지원하는 만큼, 파일을 인덱스하고 동시에 파일 크기를 통계적으로 관리하는 유사한 도구를 찾을 수 있을 것입니다.

WizTree

공식 웹사이트: https://www.diskanalyzer.com/

일반 설치 모드나 그린 버전 압축 해제 후 실행

속도가 빠르고, 데이터 표시 유형이 풍부하며, 왼쪽에 트리 형태의 모드와 오른쪽에 파일 유형이 있고, 물론 하단 막대에도 시각화된 표현 방식도 있습니다

SpaceSniffer(업데이트 2023년 이후 더 이상 업데이트되지 않습니다.)

소프트웨어 공식 웹사이트: http://www.uderzo.it/main_products/space_sniffer/

조작은 매우 간단합니다. 해당하는 디스크 문자를 선택하면 소프트웨어가 그래픽 방식으로 폴더 크기를 표시하는데, 크기가 클수록 이미지 내 해당 매트릭스도 커집니다. 나머지 조작은 직접 클릭해보면 알 수 있을 것이고, 조건 필터를 입력하여 파일을 지원합니다.

  • 파일 크기 필터링
  • 파일 날짜 필터
