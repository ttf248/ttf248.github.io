AI 프로그래밍 및 작업 분할

2년 전 사이트에 코드 영역 복사 기능을 추가했는데, 여러 날을 끙끙거린 끝에 최종 렌더링 효과가 만족스럽지 않았습니다. 반쪽짜리 프론트엔드 개발자였기 때문에 개선하려 하지는 않고 그냥 사용했습니다. 올해 AI로 미니 프로그램을 개발하면서 프론트엔드 개발에도 더 익숙해졌고, 그래서 다시 한번 (AI 재설계) 리팩토링을 했습니다.

깨달음: 과제를 분해하니 작은 작업들이 많아져서, AI 개발에 적합하네요. 이미 폐쇄된 앱인데 75%의 코드를 AI가 제공했고, 여러 개의 작업으로 분할하여 배포했으며, 사람이 프로젝트 코드 조립을 담당했습니다.

전문

간단하다고 생각했던 일이지만, 코드 복사 버튼을 추가하면 됩니다

본문

CSS와 JavaScript 코드를 설계하여 class=“highlight"인 웹 요소의 컨테이너 내에 복사 버튼을 추가하고, 코드 복사 기능을 구현합니다

제공된 첫 번째 초안 코드에는 문제가 있는 듯 보입니다. 다시 생성한 초안은 기본적으로 사용 가능하지만, 복사된 내용이 정확하지 않고 불필요한 텍스트가 포함되어 있습니다. 핵심 색상 디자인과 어떤 효과를 포함할지 수동으로 지정하여 추가 최적화를 유도했습니다.

버튼의 크기와 색상에 대한 특정 요구 사항이 있다는 것을 파악했습니다. 귀하의 요구 사항에 맞게 버튼 스타일을 다시 디자인할 수 있습니다. 다음은 제공하신 색상 구성표를 포함하고 버튼 크기를 최적화한 조정된 CSS 스타일입니다.

.copy-btn {
  position: absolute;
  top: 2px; /* 调整位置 */
  right: 2px;
  padding: 3px 6px; /* 缩小尺寸 */
  font-size: 12px; /* 减小字体大小 */
  background-color: #8FD1E1; /* 按钮背景色 */
  color: #272822; /* 文字颜色 */
  border: 1px solid #272822; /* 边框颜色 */
  border-radius: 3px; /* 圆角 */
  cursor: pointer;
  opacity: 0.7; /* 默认半透明 */
  transition: opacity 0.3s ease, background-color 0.3s ease; /* 平滑过渡效果 */
}

.copy-btn:hover {
  opacity: 1; /* 鼠标悬停时不透明 */
  background-color: #A3D4E5; /* 鼠标悬停时的背景色 */
}

.highlight {
  position: relative;
}

이 스타일에서 저는 버튼 크기와 글꼴 크기를 줄여 더욱 세련되게 만들었습니다. 동시에 버튼의 기본 상태는 반투명하며, 마우스가 버튼 위에 올라갈 때 불투명해지고 배경색이 약간 밝아져 시각적인 피드백을 제공합니다. 이러한 디자인은 귀하의 색상 요구 사항을 충족하면서도 현대적이고 사용자 친화적인 특징을 유지합니다.

해당하는 더 간단한 JavaScript를 제공합니다

document.addEventListener('DOMContentLoaded', function() {
  // 查找所有 .highlight 元素
  const highlights = document.querySelectorAll('.highlight');

  highlights.forEach(function(highlight) {
    // 创建复制按钮
    const copyBtn = document.createElement('button');
    copyBtn.textContent = '复制';
    copyBtn.classList.add('copy-btn');

    // 将按钮添加到每个 .highlight 元素中
    highlight.appendChild(copyBtn);

    // 添加点击事件监听器
    copyBtn.addEventListener('click', function() {
      // 假设 .highlight 下的第一个 <code> 元素包含要复制的代码
      const code = highlight.querySelector('code') ? highlight.querySelector('code').textContent : '';

      navigator.clipboard.writeText(code).then(() => {
        // 显示复制成功信息
        copyBtn.textContent = '已复制!';
        setTimeout(() => {
          copyBtn.textContent = '复制';
        }, 2000);
      }).catch(err => {
        console.error('复制失败: ', err);
      });
    });
  });
});

사후기

비록 코드 영역의 오른쪽 상단에 복사 버튼을 추가하는 것을 더 기대했지만, 프론트엔드에 대한 이해 부족으로 기능이 정확하게 설명되지 않았고, AI와 몇 번이고 반복했음에도 예상한 결과를 얻지 못했습니다
