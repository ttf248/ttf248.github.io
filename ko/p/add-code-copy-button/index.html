기본적인 프론트엔드 개발 지식이 부족해서 이해가 안 되는 부분은 브라우저를 __INLINE_CODE_0__BOLD_2JavaScript켜두고, 콘솔에 로그를 확인했다. 많이 모르는 부분이 있었지만 차분히 마음을 가다듬고 논리를 천천히 분해하니 결국 해결책을 찾을 수 있었다.

  1. __INLINE_CODE_0__노드가 여러 개 있는데, 여기서 말하는 것은 단일 코드 블록이고, 테마가 자체적으로 라인 넘버를 생성하여 복사 버튼이 두 개로 나타나는 현상입니다
  2. 관심 있는 주제의 코드 하이라이트 렌더링을 구현하고 싶지만, 해당 테마 설정에 익숙하지 않다
  3. 코드 하이라이트 제어 설정
  4. 설정 파일을 조정해도 계속 안 되고, 렌더링 결과가 예상과 다릅니다
  5. 이 설정들 발견하고 자료를 다시 뒤적거리며 설정을 조정해 보는데, 일단 행 번호는 제거할게요
  6. 맞춤 설정된 __INLINE_CODE_0__BOLD_2js 스크립트
  7. 이렇게까지 일을 벌였으니, 문득 최근에 봤던 색상 조합이 괜찮다는 생각이 들어서 버튼 스타일을 중국풍 천청색으로 바꿔보자
