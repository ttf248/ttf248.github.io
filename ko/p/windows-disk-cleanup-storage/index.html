어떤 버전부터인지는 정확히 기억나지 않지만, Windows 11에서 디스크 정리 도구가 크게 개선되어 훨씬 더 스마트해졌습니다

중요한 것은 공식 도구라는 점이며, 파일 삭제 오류가 없고, 광고도 없고, 팝업창도 없고, 백그라운드 프로세스도 없으며, 불필요한 기능이 전혀 없습니다

Windows 11에서 설정-시스템-저장 공간-임시 파일로 이동하여 디스크 정리 도구를 열 수 있습니다

저장 인터페이스

일반 사용자는 청소 제안을 선택하면 시스템에서 사용 상황에 따라 몇 가지 제안을 제공합니다

개발자로서 저의 디스크에는 임시 파일이 많기 때문에 임시 파일을 선택했습니다. 그 안에는 Visual Studio 임시 파일, Windows Update 임시 파일 등 다양한 임시 파일들이 포함되어 있습니다.
